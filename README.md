Trackee is a tool which periodically grabs screenshots and stores them, making users' activities tracking
much easier.

# Requirements

## Compiling

Vala compiler (> 0.10), GTK+ v3.x and GDK v3.x development headers.

## Runtime

GTK+ v3.x and GDK v3.x
